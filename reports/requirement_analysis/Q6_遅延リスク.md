# Q6: 遅延が発生する見込みがある工程はありますか

**質問**: 現在の配置でそれぞれの納期までに遅延が発生する見込みがある工程はありますか

**結論**: ✅ **完全実装可能** (追加開発不要)

**工数**: 3時間
**優先度**: 高

---

## 実装方法

Q4と同じロジックで、全業務×全工程をループして遅延チェック。

遅延見込みがある工程のみリスト表示。

## 応答例

```
⚠️ 遅延リスクのある工程が3件見つかりました

【🔴 緊急】適徴 - エントリ1
- 現在: 3名 → 必要: 5名 (あと2名)
- 遅延見込み: 15分

【🟠 要注意】あはき - 補正
- 現在: 2名 → 必要: 3名 (あと1名)
- 遅延見込み: 5分
```
